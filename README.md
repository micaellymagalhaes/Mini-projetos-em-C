# Mini Projetos em C

Este repositório reúne pequenos projetos desenvolvidos em linguagem C, com o objetivo de praticar lógica de programação, estruturas condicionais, laços de repetição, funções, vetores e structs.

## Projetos

### Calculadora

Programa simples que realiza operações matemáticas básicas:

- Soma
- Subtração
- Multiplicação
- Divisão

Arquivo:

```text
calculadora.c
```

### Jogo da Velha

Jogo da velha feito em C, utilizando matriz e lógica de verificação de vencedor.

Arquivo:

```text
jogodavelha.c
```

### Sistema de Biblioteca

Sistema simples para controle de livros, simulando um estoque de biblioteca/livraria.

Funcionalidades:

- Cadastrar livro
- Listar livros cadastrados
- Buscar livro por título
- Vender livro
- Repor estoque

Arquivo:

```text
bibiliotecaEsistema.c
```

## Tecnologias utilizadas

- Linguagem C
- GCC
- Visual Studio Code
- Git
- GitHub

## Como executar os projetos

Para compilar um arquivo em C, use o comando:

```bash
gcc nome_do_arquivo.c -o nome_do_programa
```

Depois execute com:

```bash
.\nome_do_programa.exe
```

Exemplo com o sistema de biblioteca:

```bash
gcc bibiliotecaEsistema.c -o biblioteca
.\biblioteca.exe
```
